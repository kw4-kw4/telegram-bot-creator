Инструкции по установке

  1. Скачайте архив с приложением
  2. Распакуйте архив в удобную директорию
  3. Откройте терминал и перейдите в директорию приложения
  4. Установите зависимости: npm install
  5. Создайте файл .env на основе .env.example
  6. Запустите приложение: npm start
  7. Откройте браузер и перейдите по адресу http://localhost:3000
