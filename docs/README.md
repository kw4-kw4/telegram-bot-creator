# Документация Telegram Bot Creator  ## Описание  Telegram Bot Creator - это веб-приложение, которое позволяет легко создавать и управлять Telegram-ботами без необходимости программирования.  ## Функционал  ### Основные возможности:  1. **Создание ботов**    - Создание нового бота через веб-интерфейс    - Поддержка токенов от @BotFather    - Автоматический запуск бота после создания    - Хранение ботов в базе данных SQLite  2. **Управление ботами**    - Просмотр списка созданных ботов    - Удаление (остановка) ботов    - Простой интерфейс для управления  3. **Команды ботов**    - Добавление пользовательских команд для ботов    - Настройка ответов на команды    - Просмотр списка команд бота  4. **Базовые команды**    - Автоматическая обработка команды /start    - Поддержка текстовых команд    - Расширяемая система команд  ## Установка  1. Клонируйте репозиторий  2. Установите зависимости: `npm install`  3. Создайте файл `.env` на основе `.env.example`  4. Запустите приложение: `npm start`  ## Использование  1. Откройте веб-интерфейс в браузере  2. Введите имя бота и токен от @BotFather  3. Нажмите "Создать бота"  4. Бот будет автоматически запущен и готов к использованию  5. Для добавления команд:    - Введите имя бота    - Укажите команду (например, /help)    - Введите ответ на команду    - Нажмите "Добавить команду"  ## Технические детали  - **Фронтенд**: HTML, CSS, JavaScript  - **Бэкенд**: Node.js, Express, Telegraf  - **База данных**: SQLite  - **Хранение данных**: SQLite (постоянное хранение ботов и команд)  ## Расширение функционала  Приложение можно расширить следующими возможностями:  - Добавление пользовательских команд    - Поддержка кнопок и inline-кнопок    - Обработка медиафайлов (изображения, документы)    - Поддержка различных типов сообщений  - Интеграция с базой данных    - Миграции для управления схемой базы данных    - Поддержка других СУБД (PostgreSQL, MySQL)  - Вебхуки    - Более эффективная обработка сообщений    - Поддержка высокой нагрузки  - Админ-панель    - Управление несколькими ботами    - Статистика использования    - Настройки безопасности  - Экспорт/импорт    - Резервное копирование конфигураций ботов    - Шаблоны ботов
