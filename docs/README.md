# Документация Telegram Bot Creator  ## Описание  Telegram Bot Creator - это веб-приложение, которое позволяет легко создавать и управлять Telegram-ботами без необходимости программирования.  ## Функционал  ### Основные возможности:  1. **Создание ботов**    - Создание нового бота через веб-интерфейс    - Поддержка токенов от @BotFather    - Автоматический запуск бота после создания  2. **Управление ботами**    - Просмотр списка созданных ботов    - Удаление (остановка) ботов    - Простой интерфейс для управления  3. **Базовые команды**    - Автоматическая обработка команды /start    - Возможность расширения функционала  ## Установка  1. Клонируйте репозиторий  2. Установите зависимости: `npm install`  3. Создайте файл `.env` на основе `.env.example`  4. Запустите приложение: `npm start`  ## Использование  1. Откройте веб-интерфейс в браузере  2. Введите имя бота и токен от @BotFather  3. Нажмите "Создать бота"  4. Бот будет автоматически запущен и готов к использованию  ## Технические детали  - **Фронтенд**: HTML, CSS, JavaScript  - **Бэкенд**: Node.js, Express, Telegraf  - **Хранение данных**: In-memory (в реальном приложении используйте базу данных)  ## Расширение функционала  Приложение можно расширить следующими возможностями:  - Добавление пользовательских команд  - Интеграция с базой данных для постоянного хранения  - Поддержка медиафайлов и документов  - Вебхуки для более эффективной обработки сообщений  - Админ-панель для управления несколькими ботами
